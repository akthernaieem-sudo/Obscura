# Obscura

Obscura is a secure media delivery platform built for photographers, videographers, and creators to privately share client galleries with protected previews and downloadable high-quality media.

## Live Demo

https://obscura-mu-nine.vercel.app

---

## Features

- Private gallery creation
- Secure media uploads
- Watermarked preview images
- Protected client gallery sharing
- Responsive modern UI
- Authentication system
- Cloud media storage with Supabase
- Mobile-friendly gallery experience
- Live deployment with Vercel

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Vercel
